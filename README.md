# 求生之路2 Mod工具箱

一个开源的求生之路2 Mod工具箱

## 功能

* 8角色互转
* VPK信息编辑
* 创意工坊上传
* 字体自定义
* 更多...

## 运行库

* .NET 桌面运行时 x64 6.0.13

> https://dotnet.microsoft.com/zh-cn/download/dotnet/thank-you/runtime-desktop-6.0.13-windows-x64-installer
